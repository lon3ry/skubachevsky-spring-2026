# Конспекты лекций Скубачевского А.А. по решению задач из курса «Многомерный анализ, интегралы и ряды»

В этом репозитории хранятся конспекты лекций Скубачевского А.А. по решению
задач из курса «Многомерный анализ, интегралы и ряды», читаемых во втором
семестре в МФТИ.

Актуальная версия конспектов доступна по
[ссылке](https://raw.githubusercontent.com/lon3ry/skubachevsky-spring-2026/refs/heads/master/main.pdf).

По всем вопросам можно писать мне в Telegram [@lon3ry](https://t.me/lon3ry).

## Компиляция

Для компиляции потребуется [GNU Make](https://www.gnu.org/software/make) и
[Tex Live](https://www.tug.org/texlive). Вероятно, что никаких проблем не
возникнет и при использовании других дистрибутивов LaTeX, но это не
проверялось. Для получения pdf достаточно дать команду:
```shell
make
```
